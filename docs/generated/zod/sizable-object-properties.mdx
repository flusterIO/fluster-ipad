
#### hideMathLabels

**Boolean** 


Hides the MathJax labels in all child components.


#### right

**Boolean** 


'Floats' the component to the right. This is often combined with `width` or the `sidebar` property to create sidebar layouts.


#### left

**Boolean** 


'Floats' the component to the left. This is often combined with `width` or the `sidebar` property to create sidebar layouts.


#### sidebar

**Boolean** 


A utility property that sets a responsive max-width to create sidebar like layouts on large screens while allowing for full-width when the window is smaller.

#### centerContent

**Boolean** 


Centers the content of this component's children, not the component itself.


#### border

**Boolean** 


Add a small, muted border to the object.


#### rounded

**Enum:**

One of the following:

- none
- small
- smedium
- medium
- large
- xl
- xxl
- fit
- full


Rounds the corners of the container. Use `rounded="full"` or `rounded="fit"` to create a completely circular component, useful for some image layouts.
