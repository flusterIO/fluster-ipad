## Highlighter (Hl)

A simple highlighter component that accepts an emphasis property as an optional boolean. If none is provided, the default is `highlight`.


### Example

```md
This is a <Hl success>green</Hl> highlighted component.
```

#### Output

This is a <Hl success>green</Hl> highlighted component.
