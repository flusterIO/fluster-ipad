# Quick Reference

## Syntax

Insert front matter into your note using yaml syntax seperated by 3 `---`:

```yaml
---
title: My optional note title
---
```

Insert **tags** into your note using the following syntax:

```mdx title="my_tag_demo.mdx"
This is my tag [[#myTag]] called 'myTag'.
```
