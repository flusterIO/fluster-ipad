## Underline (Ul)


````long-docs
A simple underline component that accepts an emphasis property as an optional boolean. If none is provided, the default is an underline that is the same color as the text.

Don't forget about the properties `thick` and `thin` either to create underlines of varying thickness.

````

### Example

```md
This is a <Ul success>green</Ul> underlined component.
```

#### Output

This is a <Ul success>green</Ul> underlined component.
