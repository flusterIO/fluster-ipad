## Emphasis
> Often used to draw attention to content by setting a set of grouped styles.


```long-docs
An _emphasis_ is a property that carries some styles, usually used to draw attention to content. Many components accept an 'emphasis', including the highlighter (`Hl`), the underline (`Ul`), the admonition, and more.

There also exists a `color` property for overriding the foreground color that accepts any of the _emphasis_ colors as a string, or any valid [css](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/color) color.
```
