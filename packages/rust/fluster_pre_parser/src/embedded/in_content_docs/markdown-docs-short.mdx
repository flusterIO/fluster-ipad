Here is a straightforward, quick-glance cheat sheet you can keep handy while you learn. 

### The Quick-Glance Markdown Cheat Sheet

| Element | What you type | What it looks like |
| :--- | :--- | :--- |
| **Heading 1** | `# Heading` | Large title |
| **Heading 2** | `## Heading` | Slightly smaller title |
| **Bold** | `**bold text**` | **bold text** |
| **Italic** | `*italic text*` | *italic text* |
| **Strikethrough** | `~~crossed out~~` | ~~crossed out~~ |
| **Link** | `[Google](https://google.com)` | [Google](https://google.com) |
| **Image** | `![Alt text](image.jpg)` | *(Displays the image)* |
| **Blockquote** | `> Important quote` | Block indented text |
| **Bullet List** | `- Item 1`<br/>`- Item 2` | • Item 1<br/>• Item 2 |
| **Numbered List**| `1. First`<br/>`2. Second` | 1. First<br/>2. Second |
| **Inline Code** | `` `print("Hi")` `` | `print("Hi")` |
| **Code Block** | ` ```python `<br/>`print("Hi")`<br/>` ``` ` | Syntax-highlighted block |
| **Divider** | `---` | A horizontal line |

***

**A quick note on math formatting:**
If you need to drop in some inline math, wrap it in single dollar signs like `$E = mc^2$`. For block-level math, remember to put the double dollar signs on their own separate lines, like this:

```markdown
$$
a^2 + b^2 = c^2
$$
