## Documentation

> Use just 1 `?` to trigger short documentation, use two `??` to trigger the full documentation.

Trigger documentation by placing specific keywords followed by either 1 or 2 `?`. All triggers must appear alone on a line, meaning no trailing or leading white space for specificity's sake.

| Trigger | Desc |
| ------- | ---- |
| `Docs??` | The one you probably want to remember. This will take you back here. |
| `Markdown??` | A quick markdown reference for those just getting started. |
| `Syntax??` | A reference to Fluster specific syntax. |
