## Sizable Objects


The `SizableObject` model describes a long list of properties that are applied to many components throughout Fluster. This class takes some common language terms like `small` or `smedium` and turns them into valid code.

Beware, however, that this is far from perfect. Every developer knows the challenge of centering a div, and it's difficult to write code that will look good in all circumstances without requiring _you_ to write more. Fluster does it's best to find a good balance between customizability and ease of use, but this is just the start. These components will continue to be refined to allow for a wide range of use cases.

A utility property that sets a responsive max-width to create sidebar like layouts on large screens while allowing for full-width when the window is smaller.

#### inline


Tells the container to not forcefully create a new line and instead to flow with the rest of the content. Unless you're trying to apply properties to text inside of a paragraph using the Container component this is most likely not what you are looking for. If you want text to wrap around an element, use the `right` or `left` properties paired with a desired `width`.
