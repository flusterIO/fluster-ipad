# Tabs

Organize your content with interchangable tabs.

```jsx
<Tabs>
    <Tab label="My label">
        My tab content one
    </Tab>
    <Tab label="My label 2">
        My tab content two
    </Tab>
    <Tab label="My label" 
        // This id is required because the 
        // label appears more than once!
        id="myUniqueId">
        My tab content three
    </Tab>
</Tabs>
```


<Tabs>
    <Tab label="My label">
        My tab content one
    </Tab>
    <Tab label="My label 2">
        My tab content two
    </Tab>
    <Tab label="My label" id="myUniqueId">
        My tab content three
    </Tab>
</Tabs>


### Properties

The `Tabs` component doesn't accept any _unique_ properties of it's own, but it does extend the `Sizable` and `Emphasis` structs, making both the `Emphasis??` and `Sizable??` docs apply to the parent `Tabs` component... **not** to the nested `Tab` component.
