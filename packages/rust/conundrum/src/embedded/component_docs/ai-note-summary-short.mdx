# AI Note Summary


The `AINoteSummary` component can be used to automatically generate a note summary and optionally attach it to your note in the database, making the summary appear alongside the note in the note search results.



| Property | Description |
| -------- | ----------- |
| generate | Force the component to regenerate the data. Beware however that notes with multiple AI components may take significant time to load when this property is set to true indefinitely. |
| apply | Apply the output of this component to the note's persisted summary. This will show the generated summary in your search results. |
| applyButton | A boolean or 'auto' property that indicates whether or not the 'Apply' button should be shown, allowing you to click to apply the generated content to your note as opposed to re-rendering the component and possibly generating new data to provide the 'apply' property. By default, this button will be shown whenever the generated summary does not match your persisted summary. |
