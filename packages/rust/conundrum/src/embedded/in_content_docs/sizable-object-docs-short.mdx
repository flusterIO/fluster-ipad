The `SizableObject` model describes a long list of properties that are applied to many components throughout Fluster. This class takes some common language terms like `small` or `smedium` and turns them into valid code.

Beware, however, that this is far from perfect. Every developer knows the challenge of centering a div, and it's difficult to write code that will look good in all circumstances without requiring <span className="italic">you</span> to write more. Fluster does it's best to find a good balance between customizability and ease of use, but this is just the start. These components will continue to be refined to allow for a wide range of use cases.
