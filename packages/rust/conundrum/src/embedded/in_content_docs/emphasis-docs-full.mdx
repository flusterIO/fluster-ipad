## Emphasis
> Often used to draw attention to content by setting a set of grouped styles.


An _emphasis_ is a property that carries some styles, usually used to draw attention to content. Many components accept an 'emphasis', including the highlighter (`Hl`), the underline (`Ul`), the admonition, and more.

There also exists a `color` property for overriding the foreground color that accepts any of the _emphasis_ colors as a string, or any valid  color.

And don't forget, the above component is available to you via the `Color` component. A future version of Fluster will include support for generating color gradients and all sorts of UI related workflows, as well as support for arbitrary emphasis colors.

Currently, arbitrary color support is a work in progress. The parser and type are already available in the Color component, but they have not yet been implemented in other components where it makes sense. This will change with the upcoming addition of the `Colorable` property group, which may or may not make it into the initial release.
