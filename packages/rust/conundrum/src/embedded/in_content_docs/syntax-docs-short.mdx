## Fluster Syntax

### FrontMatter

Insert front matter into your note using yaml syntax seperated by 3 `---`:

```yaml
---
title: My optional note title
---
```




### Tags 

Insert **tags** into your note using the following syntax:

```txt -- title="my_tag_demo.mdx"
This is my tag [[#myTag]] called 'myTag'.
```

### Citations



The citation key must match the bibliography entry key in your bibliography. For example:

```txt
As you can see, according to Ives & Stillwell [[cite:IvesStillwell]]...
```



### Dictionary Entries

You can insert dictionary entries in any note using the following syntax:

````txt
```dictionary -- My Dictionary Title
This is my dictionary entry.
I can insert any mdx content I like here.
```
````



### Horizontal Rule with Children

```txt
--- My title with math $\delta$ ---
```




### Title
> With an optional subtitle



```txt
# My title {#myTitleWithAnId}
> My subtitle that follows immediately on the next line
```




It is important to note though that there must be _no_ whitespace between the title and the block quote. That is what distingishes the behavior between a block quote and a subtitle.
