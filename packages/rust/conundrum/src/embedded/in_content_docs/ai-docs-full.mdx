## Fluster AI

> Please keep in mind that while Apple's on-device AI models are impressive, they are not as capable as server scale models. Fluster will integrate with server scale providers soon, but as I'm builiding this while homeless, I can't sign up for the necessary accounts to even test those capabilities. If you can, please consider supporting Fluster either directly or via sharing it with others so that we can expand Fluster's capabilities.

Now that Fluster has settled on the Apple platform we can integrate Apple's on-device AI models for various tasks.


### Capabilities

| Task | Description |
| ---- | ----------- |
| Summarize | Summarize the current note. This generates a component that can be deleted if you like, but it also appends the note's summary to your note in the database, making it appear with your note in Fluster's search results. This can be regenerated at anytime using a similar prompt. |
| Study Guide | Generate a study guide for the currently focused notes. Upcoming versions will support querying multiple notes for more complete guide generation. |


With many, _many_ more planned for the near future. Having access to first-class, on-device AI models on almost every user's device is a dream for a developer, and Fluster will take full advantage of the AI capabilities provided by Apple as the app continues to grow.
