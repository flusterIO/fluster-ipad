
## Components




| Component | Description |
| --------- | ----------- |
| `Admonition` | A card with a colored header that can be optionally foldable. Used to draw attention to important content. |
| `Hl`,<br />`Highlight` | Highlight the background of text in a variety of colors. |
| `Ul`,<br />`Underline` | Underline text in a variety of colors. |
| `Card` | A subtle layout with a title, an optional subtitle and a body. |
| `Container` | A simple, unstyled utility component that can be used to apply arbitrary styles to content. |
| `Grid` | A responsive grid that allows for making sure your notes align well on any screen size. |
| `Hr` | Almost identical to the normal markdown based horizontal rule (the `---` syntax), but this can accept a 'content' property can be used to insert a label into the divider. |
| `Hint` | Add some subtle, colored text to draw attention to smaller content. |
| `AINoteSummary` | Generate a summary of your note automatically and optionally apply it to your note's search result item. |
