## Utility Container (Container)


The `Container` component is an intentionally almost entirely unstyled component that accepts most of the _generic_ properties accepted elsewhere. This means that it takes an optional `Emphasis` as a boolean, and all of the `SizableObject` properties like `width`, `margin`, `padding`, `text` for font size and a handful of positionable props. The goal with the `Container` component is to provide a blank canvas to allow you to create some responsive & informative layouts.
