## Utility Container (Container)

> Note that the `label` property can also be changed from it's default 'Hint' to any string.

The `Container` component is an intentionally almost entirely unstyled component that accepts most of the _generic_ properties accepted elsewhere. This means that it takes an optional `Emphasis` as a boolean, and all of the `SizableObject` properties like `width`, `margin`, `padding`, `text` for font size and a handful of positionable props. The goal with the `Container` component is to provide a blank canvas to allow you to create some responsive & informative layouts that fit your own use case.

````long-docs
<Hint>
If you're studying for an exam, consider adding an `error` or `research` property to the container's you're using that are holding subjects you're still struggling with. With the change of just one word you can change the color of the container and the text within it to reflect a variety of scenarios. The _emphasis_ colors will soon be user-extendable, no longer limiting you to the provided emphasis colors either.
````
