title: Future Plans
---


Over the past few months I've been working day and night on a pair of MacOS and iPad applications to replace Fluster on those platforms, and I could not be more happy with the results... I just need to be able to afford that Apple developer license and both will be ready for release in the coming 3-4 weeks. The pair of applications are far more capable, with an entirely different approach taken to blending markdown with a Jupyter-like experience in a mobile environment.

For anyone familiar with something like Jupy-text, something similar is the goal while being completely mobile friendly. To achieve this the plan is to write a domain-specific language that will likely be very python-esk in syntax. This would allow c-python like performance in a completely cross-platform language for math, science, and note-taking related tasks, mixed in directly with your note content.
